## 🎮 How to Play

1. A random number is generated within a range
2. Enter your guess
3. Get hot/cold hints to zero in on the answer
4. Try to guess it in as few attempts as possible!

---

## 🚀 Run Locally

```bash
git clone https://github.com/dhruvalvaishnav/GuessTheNumbers.git
cd GuessTheNumbers
# Open index.html in your browser — no build step needed!
```
